## Radiofrecuencia

Todos los dispositivos inalámbricos funcionan dentro del **espectro electromagnético**.  
Las redes **WLAN** operan principalmente en dos bandas de frecuencia: **2.4 GHz** y **5 GHz**.  
Cada dispositivo inalámbrico tiene **transmisores y receptores** ajustados a frecuencias específicas de ondas de radio para comunicarse.

Las siguientes bandas se asignan a las LAN inalámbricas 802.11:

- **2.4 GHz (UHF)** → _802.11b / g / n / ax_
    
- **5 GHz (SHF)** → _802.11a / n / ac / ax_
